Financial Analytic iOS App

This is a simple iOS app I built using SwiftUI to help users track their expenses and understand their spending through a basic dashboard and analytics view.

The goal of this project was to practice building a clean UI, structuring an app properly, and working with real-world features like expense tracking and budgeting.

Features
Add and manage expenses
View all expenses in a list
Basic analytics screen for insights
Dashboard overview
Simple and clean UI using SwiftUI
Tech Used
Swift
SwiftUI
Xcode
How to Run

Clone the repo and open the .xcodeproj file in Xcode, then run it on the simulator.

Notes

This is a student project focused on learning iOS development and improving frontend and app structure skills. More features can be added in the future like data persistence, charts, and backend integration.
